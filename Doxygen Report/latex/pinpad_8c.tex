\doxysubsection{Detailed Description}
\+: Handles Pin Pad page of the UI 

\begin{DoxyAuthor}{Author}
\+: Ali Nanji 
\end{DoxyAuthor}
This file contains all the methods required to manage the Pin Pad page of the UI. The pin pad page displays a pin pad where if the user enters the correct pin, they are directed to the status page. The public functions\textquotesingle{} prototypes have been defined in \mbox{\hyperlink{pinpad_8h}{pinpad.\+h}}
